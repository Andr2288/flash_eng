# System Specification Document (SSD)

## 1. Призначення системи

FlashEng — це SPA-застосунок для вивчення англійської лексики через вправи.

---

## 2. Функціональні вимоги

### FR-01 Додавання слова

Користувач може додати слово або фразу.

### FR-02 Перевірка дубліката

Система не дозволяє додати дублікати.

### FR-03 Завантаження словника

Система отримує дані із Supabase.

### FR-04 Генерація вправи "Переклад"
