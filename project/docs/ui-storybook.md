# UI Documentation (Storybook)

## Обрані UI-компоненти

Для документації обрано два компоненти інтерфейсу системи FlashEng:

- Button
- Exercise Card

---

## Button Component

### Default

Кнопка у стандартному стані, використовується для запуску дій (наприклад, "Start Practice").

### Disabled

Кнопка неактивна, коли дія недоступна (наприклад, якщо не введені дані).

### Loading

Кнопка відображає процес виконання запиту (наприклад, генерація вправи через API).

---

## Exercise Card Component

### Default

Відображає завдання користувачу (текст вправи та поле для відповіді).

### Error

Показує повідомлення про помилку при невдалому запиті до API.

### Loading

Показує loader під час генерації вправи.

---

## Висновок

Storybook-документація дозволяє:

- ізольовано переглядати компоненти;
- тестувати різні стани UI;
- спрощує розробку та підтримку інтерфейсу.
